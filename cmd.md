# Подсказки по коммандной строке

Команда смены директории
```sh
cd c:\forder_name
```

Команда отображения текущей директории. MacOS, Linux
```sh
pwd
```

Листинг текущей директории
Windows:
```sh
dir
```
Linux, MacOS:
```sh
ls
```

Удаление файла
```sh
del <filename>
```
в Linux, MacOS:
```sh
rm <filename>
```
